# Practice Problems

## List of Questions

1. Given an integer, write a function to check if it is even or odd. Return `true` if the integer is even; otherwise `false`.

2. Given two integers - `start` and `end`, write a function to print all the integers in the range `[start, end]`.

3. Given two integers - `start` and `end`, write a function to print all the integers in the range `[start, end)`.

4. Given two integers - `start` and `end`, write a function to print all the integers in the range `(start, end]`.

5. Given two integers - `start` and `end`, write a function to print all the integers in the range `(start, end)`.

6. Given two integers `start` and `end`, write a function to print all the even integers in the range `[start, end]`.

7. Given an integer `num`, write a function to check if it is a prime number or not.

8. Given an integer between `0` to `100`, convert the integer into a letter grade. Write a function to return a string with letter grades.

9. Given an integer, write a function to print a right-angle triangle with stars. 
Example:
```
input : 3
output:
*
*	*
*	*	*
input : 5
output:
*
*	*
*	*	*
*	*	*	*
*	*	*	*	*
```

10. Given an integer, write a function to print a mirror image of the right-angle triangle from the above example.
```
input: 4
output:
            *
        *   *
    *   *   *
*   *   *   *

input: 5
output:
                *
            *   *
        *   *   *
    *   *   *   *
*   *   *   *   *

input: 3
output:
        *
    *   *
*   *   *
```

11. Given a list of integers, write a function to return the sum of all the integers in the list.

12. Given a list of integers, write a function to print all the even integers from the list.

13. Given a list of integers, write a function to print all prime numbers from the list.

14. Given a list of integers, write a function to return the largest number from the list.

15. Given a list, write a function to return the smallest number from the list.

16. Given a list of integers, write a function to return the reversed list.

17. Given a list of integers, write a function to reverse the given list.

18. Given a list of integers, write a function to move all the zeros to the front of the list.

19. Given a list of integers and an integer `key`, write a function that returns `true` if the key exists in the list; otherwise returns `falase`.

20. Given a list of integers, write a function to sort the list using bubble sort.

21. Given a list of integers, write a function to sort the list using insertion sort.

22. Given a string, write a function to sort the string.

23. Given a string, write a function to check if the string is a palindrome or not.

24. Given two strings `s1` and `s2`, write a function to check whether they are anagrams of each other.

25. Given a file path (or file name, for the sake of simplicity), write a function to read the file line by line and print it on the screen.

26. Given a file (from hereafter, this means you are given the path to a file so that you can open a file), write a function to count the number of words in the file. For this problem, a word is any alphanumeric string separated by white spaces.

27. Given a file, write a function to print the number of occurrences of each word in the file.

28. Given a list of strings, write a function to print anagrams from the list.
```
input: [‘act’, ‘abba’, ‘why’, ‘bbaa’, ‘abab’, ‘tac’, ‘hi’, ‘who’, ‘how’ ]

output:
‘act’, ‘tac’
‘abba’, ‘bbaa’, ‘abab’
‘why’
‘hi’
‘who’, ‘how’
```
